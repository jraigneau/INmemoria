h1. INmemoria

INmemoria est une application lié au site "Dignami":http://www.dignami.com pour afficher la rubrique "en mémoire d'eux".

http://labs.zeneffy.fr/projets/INmemoria

h2. Introduction

 INmemoria est simplement une interface basique d'administration permettant de d'entrer de nouveaux hommages (photos + texte) afin de les'afficher dans une iframe sur le site de "Dignami":http://www.dignami.com

Les photos sont stockées sur picasaweb via la librairie "gdata-ruby-util":http://code.google.com/p/gdata-ruby-util/ car "Heroku":http://www.heroku.com ne permet de pas de stocker des contenus et surtout picasa est gratuit (et rapide)

Les autres technos utilisées: "Sinatra":http://www.sinatrarb.org, "HAML":http://haml-lang.com/ et "Datamapper":http://www.datamapper.org
